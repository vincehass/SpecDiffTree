# Assets Directory

This directory contains visual assets for the SpecDiffTree project.

## Files

### Schematics

- **`specdifftree_schematic.png`** - Main methodology overview showing MaxEnt-TS approach

  - Shows the complete pipeline from prompt to output
  - Highlights the 4-component MaxEnt-TS search process
  - Includes tree exploration visualization (81Ã— improvement)
  - Publication-quality (300 DPI)
  - Generated by: `generate_schematic.py`

- **`schematic_overview_2.png`** - Original time-series captioning reference
  - Used as inspiration for SpecDiffTree schematic
  - Shows Text-Time-Series-Prompt (TTSP) approach

### Logos

- **`ASL_Logo2.svg`** - ASL logo (color version)
- **`ASLwhite.svg`** - ASL logo (white version)
- **`cdhi_logo.png`** - CDHI logo

## Generation Scripts

- **`generate_schematic.py`** - Python script to generate the main schematic
  - Uses matplotlib for high-quality visualization
  - Customizable colors, layout, and text
  - Run: `python assets/generate_schematic.py`

## Usage in Documentation

To include the schematic in markdown files:

```markdown
![SpecDiffTree Methodology](assets/specdifftree_schematic.png)
```
